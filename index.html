Git Quiz

What is Git and why is it so important?

Git is a version control system used by developers. It tracks any updates and changes you have made and allows collaberation with others.

What’s a branch and why do we create them?

A Branch is a new repo made from your orginal, this allows you to work on the code without making changes to your 'master code'. This acts a fail safe, so you are not breaking/disturbing your main code whilst you are changing / adding code.

What’s the working tree?

Working tree is a directory containing all the files you are working on for a specific repo

What’s the staging area?

The Staging area is the are that holds any files that are about to be commited

What’s the command to retrieve the latest version of your files from the remote serve?

git pull
